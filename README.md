# userPiss
Extensive cybersecurity tool for user generation allowing for customization of user behavior. 

## What????

userPiss is a tool for large account generation on Linux (maybe others later) that has the following features

### Account Generation:
- Random Names
- Name List (Customizable)
- Name Interations (user1, user2... user 621)
- Restricted Home Environment with customizable template
- Group Randomization 

### Account Interation:

<ins>**Passwords**</ins>
- Common Password List
- Random Passwords (Customizable)
- TOTP

<ins>**Login**</ins>
- Custom message (rejection or login)
- Login with message only
- Rejection
- IP Ban after rejection or disconnection
